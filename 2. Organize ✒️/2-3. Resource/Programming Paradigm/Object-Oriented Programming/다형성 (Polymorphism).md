다형성은 본질적으로 **"많은 형태를 가질 수 있는 능력"** 을 의미한다. 컴퓨터 과학에서는 다형성을 하나의 추상 인터페이스에 대해 코드를 작성하고 각각의 구현체를 연결할 수 있는 능력으로 정의한다. 즉, 다형성은 **여러 타입을 대상으로 동작하는 코드를 작성하는 방법**이라고 할 수 있다.
